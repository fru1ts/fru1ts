ChatGPT国内访问

Last updated on February 7, 2023 am

最近国外的ChatGPT玩得非常火爆，充分体现了AI智能的强大魅力

ChatGPT是由美国人工智能研究实验室OpenAI在2022年11月30日发布的全新聊天机器人模型，一款人工智能技术驱动的自然语言处理工具。它能够通过学习和理解人类的语言来进行对话，还能根据聊天的上下文进行互动，真正像人类一样来聊天交流，甚至能完成撰写邮件、视频脚本、文案、翻译、代码等任务。

可惜该项目却不对国内开放😭

所以记一篇国内用户如何体验ChatGPT

首先需要借助梯子才能访问到网站

然后就是点击Sign up注册：填写邮箱，邮箱验证

接着开始出现问题：需要手机号码验证，而该应用对国内手机号不支持🤮

这里可以注册一个海外的手机号码

先注册，再充值个0.2$就够了，大概1.43￥

绕后搜索openAI

可以注册一个巴西的

将电话号码填到注册的地方，然后Send code via SMS

回到购买手机号码的地方

看到短信后直接验证就搞定啦

可能有人没收到短信可以叉掉手机号码重新买一个，只要没有受到短信20分钟内退掉，会把钱退回的

接下来就是尽情玩啦

也可以用Microsoft Account或者Google,用哪个注册，下次登录的时候就要用哪种方式登录
